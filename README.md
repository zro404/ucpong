## Usage
> Originally built with Go 1.24
#### 1. Clone the repo
```bash
git clone https://github.com/zro404/ucpong.git
cd ucpong
```

#### 2. Install Go Dependencies:
```bash
go mod tidy
```

#### 3. Run the Server
```bash
go run .
```
#### visit http://localhost:8000
